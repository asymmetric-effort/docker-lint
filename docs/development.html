Development

Common tasks are managed with make:

| Target | Description | | ------ | ----------- | | all | Run clean, lint, test, and build | | bump_version | Increment the minor version and tag the current commit | | bump_version/major | Increment the major version and tag the current commit | | lint | Run static analysis | | test | Run unit and integration tests | | build | Build the docker-lint binary | | tidy | Update Go module dependencies | | clean | Remove build artifacts |
